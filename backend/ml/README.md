ML models for GuardGig

Files:
- train_and_save_models.py — generate synthetic data, train models and save pickles
- predict.py — load models and expose get_risk_score / get_fraud_score with fallbacks
- risk_model.pkl, fraud_model.pkl — model artifacts (generated by running the training script)

To train models locally:

    python backend/ml/train_and_save_models.py

This will create `risk_model.pkl` and `fraud_model.pkl` in `backend/ml`.
